# Santa Cypher

A silly encrypter that converts (poorly) between text and Santa laughing.

RETURN `HoHoHoğŸ…HoHOğŸ…HOHoğŸ…HOğŸ…HoHOğŸ…`

GIVEN `HoHoHoğŸ…HoHOğŸ…HOHoğŸ…HOğŸ…HoHOğŸ…`
